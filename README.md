# Pokédex – API Based Web App

A small web application that loads and displays Pokémon data from an external REST API.
Users can browse, search and view detailed information about individual Pokémon.

## Live Demo

Open in browser: [Pokédex](https://pokedex.sascha-gerspach.de/)

## Features

- Loading and processing Pokémon data from the PokéAPI
- Search and filter Pokémon
- Detail view with stats and images
- Dynamic rendering of data
- Loading states while data is being requested

## Technologies

- HTML
- CSS
- JavaScript
- REST API (PokéAPI)

## How to run locally

- Download or clone the repository
- Open `index.html` directly in your browser  
  (No build step or installation required)

## Notes

This project focuses on working with asynchronous JavaScript, handling API requests and rendering remote data dynamically in the UI.
